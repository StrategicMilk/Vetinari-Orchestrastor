# Vetinari Governance

This directory contains governance documents for Vetinari.

## Structure

```
governance/
├── AGENTS.md           # Agent definitions and responsibilities
├── CLAUDE.md           # Development guide
├── decisions/           # Decision log
│   └── YYYY-MM-DD.md   # Individual decision docs
└── templates/
    ├── decision.md     # Decision template
    └── policy.md       # Policy template
```

# UI

`ui/svelte` is the canonical UI source for new work.

`ui/static/svelte` is generated Svelte build output. Rebuild it from
`ui/svelte` when needed, and do not treat it as source.

`ui/legacy` contains dormant historical templates and static assets retained
for reference. Do not add new features there or mount those templates as public
routes without a license, CSP, and attribution review.

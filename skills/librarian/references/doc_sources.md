## Search Strategies

### For Library X
1. Check official docs first
2. Search GitHub for "awesome-X"
3. Look for "X examples" repo
4. Check Stack Overflow

### For API Integration
1. Official API docs
2. SDK documentation
3. Developer guides
4. Sample apps

### For Best Practices
1. Official guides
2. Industry blogs
3. Conference talks
4. Open source examples

## Citation Format

When citing sources, use:
```
[1] Official Documentation - [URL]
[2] GitHub Example - [repo/name] - [URL]
[3] Stack Overflow - [answer link]
[4] Blog Post - [title] by [author] - [URL]
```

## Version Awareness

Always note:
- Library version in use
- Docs version referenced
- Breaking changes between versions
- Recommended stable version

# Synthesis Methods

## Synthesis Techniques

### 1. Thematic Analysis
- Group by themes
- Identify patterns
- Note variations
- Extract insights

### 2. Comparison Matrix
- Create comparison table
- Add criteria
- Score each option
- Highlight differences

### 3. Timeline Synthesis
- Chronological order
- Identify dependencies
- Note milestones
- Track progress

### 4. Hierarchical Synthesis
- Top-level summary
- Supporting details
- Evidence/references
- Action items

## Report Templates

### Executive Summary
```
## [Project Name] - Executive Summary

**Status**: [On Track/At Risk/Blocked]
**Completion**: [X]%
**Key Highlights**: [3-4 bullet points]
**Risks**: [Key risks]
**Next Steps**: [Immediate actions]
```

### Decision Document
```
## Decision: [What]

**Proposed by**: [Who]
**Date**: [When]
**Decision**: [What was decided]

### Rationale
[Why this decision]

### Alternatives Considered
- Option A: [why not]
- Option B: [why not]

### Implementation
[How it will be done]

### Review Date
[When to revisit]
```

### Research Summary
```
## Research: [Topic]

### Question
[What we wanted to find]

### Methodology
[How we researched]

### Key Findings
1. Finding with source
2. Finding with source

### Recommendation
[What to do next]
```

## Quality Checklist

- [ ] Clear purpose stated
- [ ] Audience identified
- [ ] All sources accounted for
- [ ] Contradictions noted
- [ ] Confidence levels shown
- [ ] Action items clear
- [ ] Next steps defined

### Problem-Solution
1. Problem statement

4. Recommendation

### Before-After
1. Current state
2. Changes made
3. Results
4. Lessons learned

### Comparison
1. Option A description
2. Option B description
3. Comparison table
4. Recommendation with rationale
